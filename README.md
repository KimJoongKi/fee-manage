## 기록

```mysql
-- mysql 조인 쿼리 추가 조건
select 
       name, 
       nvl(sum(price),0), 
       unpaid
  from user u left outer join fee_log as f on u.name = f.contents
where u.secession_date is null
  and f.memo = ''
group by contents
order by name
```
